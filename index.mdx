SilverBullet is a programmable, private, self-hosted personal knowledge management platform. You write in Markdown, organize with bi-directional links and tags, and extend everything with Lua scripting — all running in your browser with no third-party services.

## What makes SilverBullet different

SilverBullet is more than a Markdown editor. Your notes become a queryable database — every page, task, tag, and custom attribute is automatically indexed and searchable using the built-in Lua Integrated Query language.

## Deploy it yourself

SilverBullet is fully self-hosted. Your data stays on your server — or your laptop.

<Steps>
  <Step title="Run with Docker">
    ```bash
    docker run -p 3000:3000 -v ./space:/space ghcr.io/silverbulletmd/silverbullet
    ```
  </Step>
  <Step title="Open in your browser">
    Navigate to `http://localhost:3000`. SilverBullet loads as a Progressive Web App — you can install it to your dock or home screen.
  </Step>
  <Step title="Create your first page">
    Press `Ctrl-k` (or `Cmd-k` on Mac) to open the page picker, type a name, and start writing in Markdown.
  </Step>
  <Step title="Extend with Lua">
    Add a `space-lua` code block to any page to define functions, commands, and widgets that are active across your entire space.
  </Step>
</Steps>

<Note>
  SilverBullet requires a case-sensitive file system. Linux is the recommended server OS. See the [installation guide](/install) for details on macOS and Windows.
</Note>
